## 警告说明

测试执行中有 7 个警告，主要是：
- `DeprecationWarning: 'locale.getdefaultlocale' is deprecated`
  - 位置: `src/adp_cli/i18n.py:447`
  - 原因: Python 3.15 将移除 `locale.getdefaultlocale`
  - 建议: 迁移到新的 locale API

- `PytestUnhandledThreadExceptionWarning` (多个)
  - 位置: subprocess 读取线程
  - 原因: Windows GBK 编码无法解码 UTF-8 输出
  - 影响: 集成测试的输出捕获，不影响测试结果
  - 建议: 使用 utf-8 编码调用 subprocess
